# AgentSessionManager Function Documentation

## Core Functions

### __init__(llm_service: LLMService, event_bus: EventBus, logger: logging.Logger, session_config: SessionConfig)
Initializes the session manager with required services and configuration.
Inputs:
- llm_service: Language model service for agent interactions
- event_bus: Event bus for system-wide communication
- logger: Logging instance
- session_config: Session configuration parameters

### process_message(message: str) -> Dict[str, Any]
Processes user messages and routes them to appropriate agents.
Inputs:
- message: User's input message
Outputs:
- Dictionary containing agent response with metadata

### end_interview() -> Dict[str, Any]
Ends the interview session and generates final analyses.
Outputs:
- Dictionary containing final results, coaching summary, and skill profile

### get_conversation_history() -> List[Dict[str, Any]]
Retrieves the complete conversation history.
Outputs:
- List of message dictionaries

### get_session_stats() -> Dict[str, Any]
Returns performance and usage statistics.
Outputs:
- Dictionary containing various session metrics

### reset_session()
Resets the session state and clears history.

## Internal Functions

### _get_agent(agent_type: str) -> Optional[BaseAgent]
Lazy loads agent instances.
Inputs:
- agent_type: Type of agent to load
Outputs:
- Agent instance or None if loading fails

### _get_agent_context() -> AgentContext
Prepares context for agent calls.
Outputs:
- AgentContext object with session information
